## <a id="matrix"></a> Протокол Matrix
Sfera Chat — это клиент открытого протокола Matrix.
Приложение **не хранит ваши сообщения на своих серверах** и **не управляет серверами Matrix**.
Все данные (сообщения, файлы, профили, контакты) хранятся исключительно на выбранном вами Matrix-сервере (homeserver).
Политика конфиденциальности определяется сервером, к которому вы подключаетесь.

Перед первым входом приложение показывает адрес сервера, с которым будет работать.
Sfera Chat взаимодействует только с выбранным Matrix-сервером и сервисом OpenStreetMap (для отображения карт).

Подробнее о протоколе: https://matrix.org

## <a id="database"></a> Локальная база данных
Для ускорения работы приложение использует локальный кэш сообщений и медиафайлов.
Кэш хранится в зашифрованной базе данных (SQLCipher).
Ключ для расшифровки находится только в Keychain iOS и не передаётся наружу.
При удалении приложения вся локальная база удаляется без возможности восстановления.

## <a id="encryption"></a> Шифрование
- Соединение между приложением и сервером защищено TLS.
- Сквозное шифрование (E2EE) используется в приватных чатах и защищённых комнатах на основе Olm/Megolm.
- Ключи генерируются на устройстве пользователя и не отправляются разработчикам.
- Разработчики Sfera Chat не имеют доступа к содержанию зашифрованных сообщений.

## <a id="permissions"></a> Разрешения приложения
Приложение запрашивает только необходимые разрешения:

#### Интернет
Обязательно — для работы с Matrix-сервером.

#### Уведомления
Для получения push-уведомлений о новых сообщениях.

#### Камера и микрофон
Для фото, видео, голосовых сообщений и звонков.

#### Фото/файлы
Для выбора и отправки изображений, видео и документов.

#### Геолокация (только при использовании)
Запрашивается только если пользователь вручную делится местоположением.

Никаких других разрешений приложение не использует.

## <a id="push"></a> Push-уведомления
На iOS используется только Apple Push Notification Service (APNs).
Сторонние push-сервисы (Firebase, OneSignal и т.п.) не используются.

Передаются только:
- ID события
- ID комнаты
- Количество непрочитанных сообщений

**Содержание сообщений не передаётся и к разработчикам не попадает.**

## <a id="child-safety"></a> Безопасность детей и соответствие требованиям App Store
Sfera Chat является Matrix-клиентом и не содержит встроенных серверов.
Разработчик не может удалять или фильтровать контент на сторонних серверах.

В приложении есть возможность отправить жалобу на сообщение:
1. Долгое нажатие на сообщение → «Пожаловаться»
2. Указать тип нарушения
3. Жалоба отправляется администратору выбранного сервера

Мы рекомендуем администраторам серверов соблюдать строгую модерацию.

Подробнее о модерации Matrix: https://matrix.org/docs/older/moderation/

## <a id="data-collection"></a> Сбор и использование данных
Sfera Chat **не собирает персональные данные**, кроме случаев, когда пользователь явно их вводит (например, имя, аватар, email на сервере).
Приложение может временно сохранять следующие данные для работы:
- Технические метаданные соединения (IP, время, идентификаторы сессий) — исключительно для установления соединения с сервером.
- Кэш сообщений и медиа (локально на устройстве, зашифрован).

Эти данные **не передаются разработчикам** и **не используются для аналитики или рекламы**.

## <a id="third-parties"></a> Передача данных третьим лицам
Sfera Chat не передаёт данные пользователям или третьим лицам.
Исключения:
- APNs — для доставки уведомлений
- OpenStreetMap — для отображения карт
- Matrix-сервер — технически получает только данные, необходимые для работы протокола

Все эти передачи минимальны и ограничены техническими требованиями.

## <a id="user-rights"></a> Права пользователей
Пользователь имеет право:
- Удалить локальные данные и кэш
- Отозвать доступ к геолокации
- Запросить информацию о хранящихся на сервере данных
- Попросить администратора сервера удалить свои данные
